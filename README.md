# 🌐 Language Translator (CodeAlpha Internship Project)

A modern **Python GUI-based Language Translator** built using `Tkinter`, supporting **30+ languages**, auto-detection, and text-to-speech.

---

## 🚀 Features

* 🌍 Translate between **30+ languages**
* 🔍 **Auto-detect source language**
* 🔄 **Swap languages instantly**
* ⚡ **Auto-translate mode** (real-time translation)
* 🔊 **Text-to-Speech (TTS)** support
* 📋 **Copy to clipboard**
* 🔢 **Character counter (2000 limit)**
* 🧵 **Multithreading** (smooth UI, no freezing)

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* deep-translator (Google Translate API)
* pyttsx3 (Text-to-Speech)
* threading

---

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/CodeAlpha-Language-Translator.git
cd CodeAlpha-Language-Translator
```

### 2. Install dependencies

```bash
pip install deep-translator pyttsx3
```

---

## ▶️ How to Run

```bash
python translator.py
```

---

## 🖥️ Application Features

* **Translate Button** → Manual translation
* **Auto Mode** → Translates while typing
* **Swap (⇄)** → Switch source & target language
* **Copy (⧉)** → Copy translated text
* **Listen (▶)** → Convert text to speech
